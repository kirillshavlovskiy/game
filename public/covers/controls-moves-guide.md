# Controls

Same content as `controls-moves-guide.html` (Dice of the Damned labyrinth UI).

## Stats

**Moves X/Y** (desktop) = used/pool after roll; not a button. Mobile by zoom: **Moves**, **Jumps**. **Menu** pauses. **End turn** skips rest; blocked in combat, teleport pick, slingshot.

## Move

0 moves → dice modal. Tap walkable cells; jump glow if you have jumps. Desktop **Controls** pad + arrow keys. Mobile ArrowUp/Down opens/closes dock. Traps/web cost extra. **−** **+** zoom ~50–200%.

## Items

Desktop: one tap per chip. Mobile: swipe dock up, pick row, **Use**. **Magic portal** on magic cells. Bomb: 1 move on map if moves left; free in fight. Teleport gem: use then tap highlight. Picker: Cancel / Random.

## Slingshot

Drag-release opposite pull; Cancel or walk off with moves.

## Fight

Skills: shield, dice reroll, holy. Mobile **Dice+Use** = same reroll toggle. Teleport/reveal/heal/torch inventory off in combat.

## Legend

Hover desktop; icon titles on traps, bombs, magic, sling, web, monsters.
